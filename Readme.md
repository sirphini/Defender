# Defender – Modellseite

## Projektbeschreibung

Das Projekt "Defender" ist eine moderne Modellseite für das Automodell Land Rover Defender. Ziel der Website ist es, potenziellen Kunden eine übersichtliche und ansprechende Präsentation der verschiedenen Fahrzeugmodelle zu bieten. Die Seite richtet sich an Autokäufer im Alter von 18 bis 50 Jahren und kombiniert Design, Information und Interaktivität.

Im Zentrum der Website stehen die Modellübersicht sowie ein interaktiver Konfigurator. Nutzer können zwischen verschiedenen Defender-Modellen (90, 110, 130) wählen, Farben anpassen und optionales Zubehör hinzufügen. Der Preis wird dabei dynamisch berechnet und direkt angezeigt. Zusätzlich bietet die Website eine Vergleichstabelle der Modelle, eine Bildergalerie, aktuelle News sowie ein Formular zur Anfrage einer Probefahrt.

Beim Design habe ich mich am Look & Feel einer echten Herstellerseite orientiert, um eine möglichst realistische und hochwertige Darstellung zu erreichen. Ziel war es, ein modernes UI zu gestalten, wie sie auch auf offiziellen Marken Websites verwendet wird.

Die Website wurde vollständig mit HTML, CSS und JavaScript entwickelt und läuft lokal im Browser. Besondere Features sind der Konfigurator mit mehreren Auswahlmöglichkeiten sowie die Umsetzung eines Dark Modes über Systempräferenzen, anstelle eines klassischen Schalters.

---

## Features

* Modellübersicht (Defender 90 / 110 / 130)
* Interaktiver Fahrzeug-Konfigurator
* Dynamische Preisberechnung
* Vergleichstabelle der Modelle
* Image Gallery
* News-Bereich
* Probefahrt-Anfrageformular
* Responsive Navigation (inkl. Burger-Menü)
* Dark Mode (systembasiert)
